HOW TO USE
================================================================================

1. Open sop-portal.jsx
2. Find: const sopData = [
3. Replace placeholder File IDs with your Google Drive IDs
4. Add all 12 SOPs
5. Save and deploy

To get Google Drive File IDs:
  - Open PDF in Google Drive
  - Copy File ID from URL: https://drive.google.com/file/d/[FILE_ID]/view
  - Paste into sopData array
